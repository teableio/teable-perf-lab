# table-delete/50k-20f

## Goal

Scale the archived mixed table from 10,000 to 50,000 rows.

## Seed Phase

Build and validate a deterministic 50,000-row, 20-field table.

## Execute Phase

Archive the table for ten isolated samples and assert V1/V2 routing and trash state.

## Primary Metric

- `deleteTableP95Ms`: p95 archive-request latency, maximum 2,000 ms.
