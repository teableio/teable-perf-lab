# duplicate-table/50k-20f

## Goal

Measure one duplicate-table request for a 50,000-record mixed 20-field table.
Compared with `duplicate-table/10k-20f`, only the source record count changes.

## Seed Phase

Build the same deterministic mixed 20-field source table in 1,000-row batches,
scaled from 10,000 to 50,000 rows.

## Execute Phase

Duplicate the table once with records included and assert V1/V2 routing.

## Primary Metric

- `duplicateTableRequestMs`: duplicate endpoint request time only; initial
  `maxMs` is 60,000.

## Verification

Wait until all 50,000 rows are available, scan the complete copy, and verify
rows 1, 25,000, and 50,000 across all fields.
