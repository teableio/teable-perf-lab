# table-restore/50k-20f

## Goal

Scale the restored mixed table from 10,000 to 50,000 rows.

## Seed Phase

Build and validate a deterministic 50,000-row, 20-field table.

## Execute Phase

Archive and restore the table for ten samples, assert V1/V2 routing, then verify the complete restored state.

## Primary Metric

- `restoreTableP95Ms`: p95 restore-request latency, maximum 2,000 ms.
